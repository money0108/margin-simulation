# 保證金對沖說明與計算範例

## 一、制度概述

本制度以固定槓桿計算分邊 Base IM，並以 Base IM 判定大小邊；對沖折減僅適用於小邊，依三產業桶與 3M 加權累積報酬率決定折減率（50% 或 20%）；0050/0056 ETF 採 look-through，成份股完全對沖部分可 100% 減收。

---

## 二、槓桿倍數規則

| 標的類型 | 槓桿倍數 | 說明 |
|---------|:-------:|------|
| ETF（月均量 ≥ 10,000 張） | 7x | 如 0050（月均量 ~97,919 張） |
| ETF（月均量 < 10,000 張） | 5x | 月均量不足萬張之 ETF |
| 股期標的（股期槓桿 > 7） | 5x | 如台積電（股期槓桿 7.41） |
| 股期標的（股期槓桿 > 6 且 ≤ 7） | 4x | 如鴻海（股期槓桿 6.17） |
| 股期標的（股期槓桿 ≤ 6） | 3x | 如廣達（股期槓桿 4.94） |
| 其他股票 | 3x | 非股期標的之股票 |

> **ETF 槓桿依月均量判定**：月均成交量 ≥ 10,000 張 → 7x，< 10,000 張 → 5x。ETF look-through 後成分股沿用母 ETF 的槓桿倍數。
>
> **股期動態對應**：股期標的的遠期槓桿倍數依「期貨槓桿倍數_完整表.xlsx」中的股期槓桿倍數判定，同一標的若有多筆期貨（如一般型＋小型），取最大股期槓桿倍數。

## 三、保證金計算公式

### 3.1 Base IM（基礎初始保證金）

```
Base IM = MV ÷ Leverage（逐檔計算，不可用平均槓桿）
```

- **MV（Market Value）** = 股數 × 當日收盤價
- 分別計算多方與空方的 Base IM

### 3.2 大小邊判定

```
若 Base_IM_Long ≥ Base_IM_Short → 大邊=LONG, 小邊=SHORT
若 Base_IM_Long < Base_IM_Short → 大邊=SHORT, 小邊=LONG
```

### 3.3 當日 IM

```
IM_today = IM_big + IM_small_after_discount
```

- **IM_big**：大邊 Base IM（不折減）
- **IM_small_after_discount**：小邊 Base IM 扣除各項折減

### 3.4 維持保證金（MM）

```
MM_today = 0.70 × IM_today
```

### 3.5 追繳判定

```
若 Equity < MM_today → 觸發追繳，需補繳至 IM_today
追繳金額 = IM_today - Equity
```

## 四、對沖折減規則

### 4.1 產業桶分類

| 產業桶 | 包含產業 |
|-------|---------|
| 電子 | 半導體、電子零組件、電腦週邊、光電、通訊網路等 |
| 金融 | 銀行、保險、金控、證券等 |
| 非金電 | 除電子、金融外之其它產業 |

### 4.2 折減類型與比例

| 對沖類型 | 條件 | 折減率 | 說明 |
|---------|------|:------:|------|
| ETF 完全對沖 | 成分股有對向曝險 | **100%** | ETF look-through 後，成分股多空完全配對 |
| 同桶對沖 | 3M 報酬差 ≥ 10% | **50%** | 同產業桶內多空對沖，報酬差異大 |
| 同桶對沖 | 3M 報酬差 < 10% | **20%** | 同產業桶內多空對沖，報酬差異小 |
| 跨桶對沖 | - | **20%** | 不同產業桶間的多空對沖 |

### 4.3 折減計算原則

1. **折減只作用於小邊**：大邊 Base IM 不折減
2. **ETF 100% 折減優先**：先計算 ETF look-through 的完全對沖
3. **同桶折減次之**：依 3M 報酬率差判定 50% 或 20%
4. **跨桶折減最後**：對小邊未被同桶覆蓋的部分給予 20%

### 4.4 3M 報酬率計算

```
單檔報酬：R_i(3M) = (P_end / P_start) - 1
加權報酬：R_bucket,side(3M) = Σ(w_i × R_i)
報酬率差：R_diff = R_big_side - R_small_side
```

## 五、ETF Look-through 機制

### 5.1 概念說明

當持有 0050 或 0056 ETF 時，系統會將 ETF 的名目市值依成份股權重拆解至個別成分股。

### 5.2 完全對沖認定

若 ETF 拆解後的成分股與直接持有的個股形成多空對沖，該配對部分可享 **100% 減收**。

```
配對金額 = min(成分股多方MV, 成分股空方MV)
100% 折減 IM = 配對金額 ÷ 槓桿
```

## 六、計算範例：一籃子股票多 vs 0050 空

### 6.1 部位設定

假設持有以下部位：

**多方部位（一籃子股票）：**

| 代碼 | 股票名稱 | 股數 | 股價 | MV | 產業桶 | 槓桿 | 槓桿來源 |
|:----:|---------|-----:|-----:|---:|:------:|:----:|---------|
| 2330 | 台積電 | 10,000 | 580 | 5,800,000 | 電子 | 5x | 股期槓桿 7.41 > 7 |
| 2317 | 鴻海 | 20,000 | 105 | 2,100,000 | 電子 | 4x | 股期槓桿 6.17, >6 且 ≤7 |
| 2882 | 國泰金 | 30,000 | 45 | 1,350,000 | 金融 | 5x | 股期槓桿 7.41 > 7 |
| 1301 | 台塑 | 15,000 | 85 | 1,275,000 | 非金電 | 5x | 股期槓桿 7.41 > 7 |

**空方部位（0050 ETF）：**

| 代碼 | 名稱 | 股數 | 股價 | MV | 槓桿 |
|:----:|-----|-----:|-----:|---:|:----:|
| 0050 | 元大台灣50 | 80,000 | 140 | 11,200,000 | 7x |

### 6.2 Step 1：計算多方 Base IM

| 代碼 | MV | 槓桿 | Base IM |
|:----:|---:|:----:|--------:|
| 2330 | 5,800,000 | 5x | 1,160,000 |
| 2317 | 2,100,000 | 4x | 525,000 |
| 2882 | 1,350,000 | 5x | 270,000 |
| 1301 | 1,275,000 | 5x | 255,000 |
| **合計** | **10,525,000** | - | **2,210,000** |

### 6.3 Step 2：ETF Look-through 拆解

假設 0050 成分股權重如下（簡化），0050 月均量 ~97,919 張 ≥ 10,000 張 → 槓桿 **7x**，ETF look-through 後成分股沿用 **7x** 計算：

| 成分股 | 權重 | 拆解後 MV | 槓桿 | Base IM |
|:------:|-----:|----------:|:----:|--------:|
| 2330 台積電 | 45% | 5,040,000 | 7x | 720,000 |
| 2317 鴻海 | 5% | 560,000 | 7x | 80,000 |
| 2882 國泰金 | 3% | 336,000 | 7x | 48,000 |
| 其他電子股 | 35% | 3,920,000 | 7x | 560,000 |
| 其他金融股 | 7% | 784,000 | 7x | 112,000 |
| 其他非金電 | 5% | 560,000 | 7x | 80,000 |
| **合計** | **100%** | **11,200,000** | - | **1,600,000** |

### 6.4 Step 3：大小邊判定

```
多方 Base IM = 2,210,000
空方 Base IM = 1,600,000

∵ 2,210,000 > 1,600,000
∴ 大邊 = LONG（多方），小邊 = SHORT（空方）
```

### 6.5 Step 4：計算 ETF 100% 完全對沖折減

檢視小邊（空方 ETF 拆解）與大邊（多方）是否有配對：

| 代碼 | 大邊 MV | ETF 拆解 MV | 配對 MV | 100% 折減 IM |
|:----:|--------:|------------:|--------:|-------------:|
| 2330 | 5,800,000 | 5,040,000 | 5,040,000 | 720,000 |
| 2317 | 2,100,000 | 560,000 | 560,000 | 80,000 |
| 2882 | 1,350,000 | 336,000 | 336,000 | 48,000 |
| **合計** | - | - | **5,936,000** | **848,000** |

**說明：**
- 2330：大邊 580 萬，ETF 拆解 504 萬，取小者 504 萬配對
- 配對部分的 Base IM = 504 萬 ÷ 7 = 72 萬，100% 折減

### 6.6 Step 5：計算同桶折減

ETF 100% 折減後，小邊（空方 ETF 拆解）殘餘：

| 成分股 | 原 MV | 已配對 | 殘餘 MV | 產業桶 |
|:------:|------:|-------:|--------:|:------:|
| 2330 | 5,040,000 | 5,040,000 | 0 | 電子 |
| 2317 | 560,000 | 560,000 | 0 | 電子 |
| 2882 | 336,000 | 336,000 | 0 | 金融 |
| 其他電子股 | 3,920,000 | 0 | 3,920,000 | 電子 |
| 其他金融股 | 784,000 | 0 | 784,000 | 金融 |
| 其他非金電 | 560,000 | 0 | 560,000 | 非金電 |

**產業桶彙總（殘餘）：**

| 產業桶 | 小邊殘餘 MV | 大邊 MV | 可對沖比例 |
|:------:|------------:|--------:|:----------:|
| 電子 | 3,920,000 | 7,900,000 | 100% |
| 金融 | 784,000 | 1,350,000 | 100% |
| 非金電 | 560,000 | 1,275,000 | 100% |

假設 3M 報酬率差皆 < 10%，同桶折減率 = 20%：

| 產業桶 | 小邊殘餘 Base IM | 可對沖比例 | 有效折減率 | 同桶折減 |
|:------:|----------------:|:----------:|:----------:|---------:|
| 電子 | 560,000 | 100% | 20% | 112,000 |
| 金融 | 112,000 | 100% | 20% | 22,400 |
| 非金電 | 80,000 | 100% | 20% | 16,000 |
| **合計** | - | - | - | **150,400** |

### 6.7 Step 6：計算跨桶折減

各桶可對沖比例皆為 100%，小邊已完全被同桶覆蓋，無殘餘需跨桶折減：

| 產業桶 | 未覆蓋比例 | 未覆蓋 Base IM | 跨桶折減 |
|:------:|:----------:|---------------:|---------:|
| 電子 | 0% | 0 | 0 |
| 金融 | 0% | 0 | 0 |
| 非金電 | 0% | 0 | 0 |
| **合計** | - | - | **0** |

### 6.8 Step 7：彙總計算 IM_today

```
折減彙總：
├─ ETF 100% 折減：      848,000
├─ 同桶折減：            150,400
└─ 跨桶折減：                  0
   ─────────────────────────
   總折減：             998,400

小邊 IM（折減後）= 1,600,000 - 998,400 = 601,600
大邊 IM（不折減）= 2,210,000

IM_today = 2,210,000 + 601,600 = 2,811,600
```

### 6.9 Step 8：計算維持保證金與追繳判定

```
MM_today = 0.70 × 2,811,600 = 1,968,120

若 Equity = 3,000,000：
  ∵ 3,000,000 > 1,968,120
  ∴ 不觸發追繳

若 Equity = 1,500,000：
  ∵ 1,500,000 < 1,968,120
  ∴ 觸發追繳
  追繳金額 = 2,811,600 - 1,500,000 = 1,311,600
```

### 6.10 計算摘要表

| 項目 | 金額 |
|------|-----:|
| 多方 MV | 10,525,000 |
| 空方 MV | 11,200,000 |
| 多方 Base IM | 2,210,000 |
| 空方 Base IM | 1,600,000 |
| 大邊 | LONG |
| 小邊 | SHORT |
| IM_big（不折減） | 2,210,000 |
| IM_small（折減前） | 1,600,000 |
| ETF 100% 折減 | 848,000 |
| 同桶折減 | 150,400 |
| 跨桶折減 | 0 |
| **總折減** | **998,400** |
| IM_small（折減後） | 601,600 |
| **IM_today** | **2,811,600** |
| MM_today（70%） | 1,968,120 |
| Gross 槓桿（有折減） | 7.73x |
| 無折減槓桿 | 5.70x |

## 七、對沖效益分析

### 7.1 折減效益

```
無折減時 IM = 2,210,000 + 1,600,000 = 3,810,000
有折減後 IM = 2,811,600

折減效益 = 3,810,000 - 2,811,600 = 998,400
折減比例 = 998,400 / 3,810,000 = 26.2%
```

### 7.2 槓桿提升

```
無折減槓桿 = (10,525,000 + 11,200,000) / 3,810,000 = 5.70x
有折減槓桿 = (10,525,000 + 11,200,000) / 2,811,600 = 7.73x

槓桿提升 = 7.73 / 5.70 = 1.36 倍
```

## 八、重要注意事項

1. **ETF 權重變動**：0050/0056 成分股權重會定期調整，需使用最新權重
2. **3M 報酬率資料**：需有足夠歷史資料（約 63 個交易日）
3. **缺碼處理**：若股票缺少產業分類，以保守口徑歸入「非金電」
4. **張數轉換**：1 張 = 1,000 股
5. **金額精度**：計算結果四捨五入至元

| 策略 | 多方 | 空方 | 預期折減 |
